## What It Does

Proofline takes messy product state, checks it against a stronger reference, tests whether readiness evidence is real enough to trust, and turns the result into a usable release-readiness route.

## What the Owner Gets

- clarity before release
- what already holds
- what needs fixing
- where the agent is masking uncertainty
- where the product still fails the check
