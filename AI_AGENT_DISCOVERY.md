# AI Agent Discovery - AIS FLOWS / Proofline

## Product Type

5-skill product readiness pack for agent chats.

## Use This When

Use Proofline when an agent needs to check a product folder, repo, workspace, or release packet before treating it as ready.

## What It Does

```text
product material -> reference gap -> readiness gate -> reality QA -> lean pass -> owner report + agent report
```

Proofline turns messy product state into a clear owner report and structured agent/system output.

## Expected Input

```text
Product folder, repo, app workspace, URL, release packet, or explicit product evidence bundle.
```

## Expected Output

```text
owner-readable report
agent/system report
evidence ledger
runtime approval list
repair feed
readiness verdict
```

## Packaged Skills

- `ais-flows-product-reference-gap-gate`
- `ais-flows-ready-gate`
- `ais-flows-reality-qa`
- `ais-flows-lean-pass`
